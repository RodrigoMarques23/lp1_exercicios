## Pergunta

48 - Indica o que é impresso no ecrã pelo seguinte código. Justifica a tua resposta,
	explicando em que consiste a operação efetuada pelo método GCD().

```cs
static void Main(string[] args)
{
    ulong a = Convert.ToUInt64(args[0]);
    ulong b = Convert.ToUInt64(args[1]);
    Console.WriteLine($"GCD between {a} and {b} is {GCD(a, b)}");
}

static ulong GCD(ulong a, ulong b) => b == 0 ? a : GCD(b, a % b);
```

## Soluções

### Solução 1

	A operção efetuada pelo método GCD, funciona como um *if* e *else if* em que se
	b == 0 então retorna a, caso contrário retorna b no lugar de *ulong a* 
	e retorna o resto de divisão entre a e b no lugar de *ulong b*.
	
	O facto de as variáveis serem sem sinal excluí o facto dos números poderem ser 
	negativos.

	Assim sendo o que é impresso no ecrã é o seguinte: 
		"GCD between a and b is  GCD(a,a)"
